2 ЗАВДАННЯ
– Розробити PHP-скрипт для реалізації роботи інтернет-магазину.
– Реалізувати сторінки: каталог товарів, кошик.
– Реалізувати додавання товарів до кошика з використанням механізму сесій.
– Виводити обрані товари в кошику у вигляді таблиці з можливістю видалення позицій.

3 ОПИС ВИКОНАНОЇ РОБОТИ  
– Розроблено структуру проєкту з двома основними сторінками: Index.php та basket.php.
– В index.php реалізовано HTML-форму для вибору кількості кожного товару і передачі даних методом POST.
– Сервер обробляє введені значення, перевіряє їхню валідність, і у разі коректності зберігає їх у сесію.
– У basket.php з сесії виводиться таблиця з переліком обраних товарів, цінами, кількістю і підсумковою вартістю.
– Реалізовано перевірку на відсутність товарів у кошику — виводиться повідомлення і посилання на сторінку покупок.

4 ВИСНОВКИ  
У результаті виконання лабораторної роботи було створено простий Web-магазин на PHP. Отримано навички роботи з HTML-формами, обробки POST-запитів та збереження даних у сесіях. Реалізоване рішення відповідає вимогам лабораторної роботи та демонструє базову логіку інтернет-магазину.

В.1 Код сторінки index.php
<?php
session_start();

$products = [
    ['id' => 1, 'name' => 'Fanta', 'price' => 1],
    ['id' => 2, 'name' => 'Sprite', 'price' => 1],
    ['id' => 3, 'name' => 'Nuts', 'price' => 2],
];

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    foreach ($products as $product) {
        $id = $product['id'];
        $count = isset($_POST["count_$id"]) ? (int)$_POST["count_$id"] : 0;
        if ($count > 0) {
            $_SESSION['cart'][$id] = [
                'id' => $id,
                'name' => $product['name'],
                'price' => $product['price'],
                'count' => $count
            ];
        }
    }
    header('Location: basket.php');
    exit;
}
?>

<!DOCTYPE html>
<html>
<head>
    <title>Продукти</title>
</head>
<body>
    <h1>Сторінка товарів</h1>
    <form method="POST" action="index.php">
        <table>
            <tr><th>Назва</th><th>Ціна</th><th>Кількість</th></tr>
            <?php foreach ($products as $product): ?>
                <tr>
                    <td><?= htmlspecialchars($product['name']) ?></td>
                    <td>$<?= $product['price'] ?></td>
                    <td>
                        <input type="number" name="count_<?= $product['id'] ?>" min="0" value="0">
                    </td>
                </tr>
            <?php endforeach; ?>
        </table>
        <br>
        <button type="submit">Купити</button>
    </form>
    <footer><a href="basket.php">Перейти до кошика</a></footer>
</body>
</html>

В.2 Код сторінки basket.php
<?php
session_start();
if (!isset($_SESSION['cart']) || empty($_SESSION['cart'])) {
    echo '<a href="products.php">Перейти до покупок</a>';
    exit;
}
echo "<table>";
echo "<tr><th>ID</th><th>Назва</th><th>Ціна</th><th>Кількість</th><th>Сума</th></tr>";
$id = 1;
$total = 0;
foreach ($_SESSION['cart'] as $name => $info) {
    $sum = $info['price'] * $info['count'];
    echo "<tr><td>$id</td><td>$name</td><td>{$info['price']}</td><td>{$info['count']}</td><td>$sum</td></tr>";
    $total += $sum;
    $id++;
}
echo "<tr><td colspan='4'>Разом:</td><td>$total</td></tr>";
echo "</table>";
?>
